# Diastema
Diastema is a [Laspi](https://www.laspi.gr/)-inspired video game, modelled after the space shooter concept.

# Language
Hellenic (el-GR)

# Installation
Just run the .exe file.

# Controls
Movement with arrows or WASD.<br/>Press 'SPACE' to pause.
